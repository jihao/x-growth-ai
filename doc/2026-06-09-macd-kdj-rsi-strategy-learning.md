# MACD / KDJ / RSI 技术指标交易策略新人入门

> 学习资料，生成日期：2026-06-09。本文只用于理解技术指标、交易策略和回测报告，不构成任何投资建议。

## 1. 先建立一个正确认知

MACD、KDJ、RSI 都是从历史价格计算出来的技术指标。它们能帮助你观察趋势、动能和短期情绪，但不能保证预测未来。

新手最容易把技术指标当成“买卖按钮”：

```text
金叉就买
死叉就卖
超卖就抄底
超买就逃顶
```

这很危险。更稳妥的理解是：

```text
指标 = 观察工具
策略 = 把观察工具转成明确规则
回测 = 用历史数据检验规则表现
风控 = 控制规则失效时的损失
```

也就是说，指标本身不是策略。只有当你明确规定“什么情况买、什么情况卖、用多少钱买、按什么价格成交、如何计算成本”时，它才变成一个可回测的交易策略。

## 2. 本文对应的回测模型

当前项目里的技术指标回测使用一套比较保守、适合入门理解的模型。

默认设定：

- 数据源：BaoStock 日 K 线。
- 价格：前复权日 K。
- 股票池：`backend/config/watchlist.json`。
- 策略：MACD、KDJ、RSI 三套策略分别独立回测。
- 信号生成：用当天收盘后的指标状态生成买卖信号。
- 成交时间：信号出现后的下一个交易日开盘成交。
- 仓位：单股票、单策略、单仓位。
- 买入：尽量满仓，但必须按 A 股 100 股整数手买入。
- 卖出：触发卖出信号时全仓卖出。
- 成本：佣金和卖出印花税都会计入。

为什么要“次日开盘成交”？

因为当天的技术指标通常要等收盘价确定后才能完整计算。如果当天收盘后才知道信号，却假设当天收盘已经买入，就容易产生未来函数。次日开盘成交更贴近真实可执行场景。

## 3. MACD 策略

### 3.1 MACD 是什么

MACD 主要观察趋势和动能。

它由三部分组成：

```text
DIF
DEA
MACD 柱
```

常见参数是 `12, 26, 9`。

计算逻辑：

```text
DIF = EMA(12) - EMA(26)
DEA = DIF 的 9 日 EMA
MACD 柱 = 2 × (DIF - DEA)
```

其中 EMA 是指数移动平均线，比普通均线更重视最近价格。

直观理解：

- `DIF > DEA`：短期动能相对更强。
- `DIF < DEA`：短期动能相对更弱。
- `DIF` 上穿 `DEA`：动能由弱转强。
- `DIF` 下穿 `DEA`：动能由强转弱。

### 3.2 MACD 入门策略规则

当前回测使用的是最经典的金叉/死叉规则：

```text
买入：DIF 上穿 DEA
卖出：DIF 下穿 DEA
```

更具体地说：

```text
昨日 DIF <= 昨日 DEA
今日 DIF > 今日 DEA
= 产生买入信号
```

```text
昨日 DIF >= 昨日 DEA
今日 DIF < 今日 DEA
= 产生卖出信号
```

如果今天收盘后出现买入信号，回测会在下一个交易日开盘尝试买入。

### 3.3 MACD 策略适合观察什么

MACD 更偏趋势跟随。它适合观察一段行情是否正在从弱转强，或者上涨动能是否开始衰减。

优点：

- 规则简单，容易理解。
- 对趋势行情比较友好。
- 不会像短线震荡指标那样过度敏感。

缺点：

- 指标滞后，买点和卖点通常不会特别早。
- 在震荡行情里容易反复金叉死叉，导致频繁交易。
- 如果趋势已经走完，金叉可能只是反弹尾声。

### 3.4 新手读 MACD 回测时看什么

重点看：

- 交易次数是否过多。
- 胜率是否稳定。
- 最大回撤是否能接受。
- 是否跑赢买入持有。

如果 MACD 交易很多，但总收益不高，说明它可能被震荡行情反复消耗。

如果 MACD 收益为负，但超额收益为正，说明它虽然亏钱，但比一直拿着少亏。

## 4. KDJ 策略

### 4.1 KDJ 是什么

KDJ 主要观察短期价格在最近区间中的位置，常用于判断短线超买或超卖。

第一步计算 RSV：

```text
RSV = (今日收盘价 - 最近 N 日最低价)
      / (最近 N 日最高价 - 最近 N 日最低价)
      × 100
```

常见 `N = 9`。

然后计算 K、D、J：

```text
K = 前一日 K × 2/3 + 今日 RSV × 1/3
D = 前一日 D × 2/3 + 今日 K × 1/3
J = 3K - 2D
```

如果没有前一日 K、D，通常初始值设为 50。

直观理解：

- K、D 越高，说明价格越靠近近期高位。
- K、D 越低，说明价格越靠近近期低位。
- J 最敏感，可能高于 100，也可能低于 0。

### 4.2 KDJ 入门策略规则

当前回测使用的是“低位金叉买入，高位死叉卖出”：

```text
买入：K 上穿 D，且 K/D 至少有一个处于 30 以下
卖出：K 下穿 D，且 K/D 至少有一个处于 70 以上
```

更具体地说：

```text
昨日 K <= 昨日 D
今日 K > 今日 D
并且 min(K, D) < 30
= 产生买入信号
```

```text
昨日 K >= 昨日 D
今日 K < 今日 D
并且 max(K, D) > 70
= 产生卖出信号
```

为什么不是所有金叉都买？

因为 KDJ 很敏感。如果不加低位条件，可能在高位也频繁金叉，容易追高。加入 `< 30` 是为了让策略更偏向“短期低位转强”。

为什么不是所有死叉都卖？

因为低位死叉可能已经跌了很多。加入 `> 70` 是为了让策略更偏向“短期高位转弱”。

### 4.3 KDJ 策略适合观察什么

KDJ 更偏短线节奏。它适合观察：

- 短期是否跌得过深后开始转强。
- 短期是否涨得过热后开始转弱。
- 震荡行情中的波段机会。

优点：

- 对短期变化敏感。
- 比 MACD 更容易捕捉较早的反弹信号。
- 在震荡行情中可能比趋势指标更灵活。

缺点：

- 在单边上涨中可能过早提示高位卖出。
- 在单边下跌中可能频繁提示低位买入。
- 假信号较多，必须配合回测和风控。

### 4.4 新手读 KDJ 回测时看什么

重点看：

- 买入后是否经常继续下跌。
- 持仓时间是否过长。
- 高位卖出是否真的保护了收益。
- 胜率高时，单次亏损是否也很大。

KDJ 策略的胜率可能看起来不错，但如果最后一笔期末持仓浮亏很大，说明它在趋势破坏时可能卖得不够及时。

## 5. RSI 策略

### 5.1 RSI 是什么

RSI 用来观察上涨力量和下跌力量的相对强弱。

常见参数是 `14`。

简化理解：

```text
RSI = 100 × 平均上涨幅度 / (平均上涨幅度 + 平均下跌幅度)
```

当前实现使用 Wilder 平滑方式计算 `RSI14`。

常见区间：

```text
RSI > 70：短期可能偏热
RSI < 30：短期可能偏冷
RSI 在 50 上方：多方力量相对更强
RSI 在 50 下方：空方力量相对更强
```

### 5.2 RSI 入门策略规则

当前回测使用的是“超卖回升买入，超买回落卖出”：

```text
买入：RSI14 从 30 以下上穿 30
卖出：RSI14 从 70 以上下穿 70
```

更具体地说：

```text
昨日 RSI14 <= 30
今日 RSI14 > 30
= 产生买入信号
```

```text
昨日 RSI14 >= 70
今日 RSI14 < 70
= 产生卖出信号
```

为什么不是 RSI 低于 30 就买？

因为低于 30 只说明短期偏弱，不代表已经开始修复。等它重新上穿 30，是为了确认下跌动能有所缓和。

为什么不是 RSI 高于 70 就卖？

因为强势行情里 RSI 可以长时间保持高位。等它跌回 70 以下，是为了确认短期强势有所降温。

### 5.3 RSI 策略适合观察什么

RSI 更适合观察多空力量变化，尤其是极端情绪后的修复。

优点：

- 规则容易理解。
- 交易频率通常比 KDJ 更低。
- 对极端超卖后的修复比较直观。

缺点：

- 可能长时间没有信号。
- 如果买入后只是弱反弹，容易被套。
- 在强趋势行情里可能卖得偏晚或买得偏少。

### 5.4 新手读 RSI 回测时看什么

重点看：

- 交易次数是不是太少，样本是否足够。
- 持仓周期是不是特别长。
- RSI 买入后有没有真正脱离弱势。
- 策略收益是否只是来自一两笔大交易。

如果 RSI 胜率显示 100%，但只有 1 笔交易，不要过度兴奋。样本太少时，胜率参考价值有限。

## 6. 买入持有和超额收益

回测报告里有两个容易混淆的指标：

```text
策略收益
买入持有
超额
```

策略收益表示按技术指标规则交易后的结果。

买入持有表示：

```text
回测开始日买入这只股票
一直持有到回测结束日
```

超额收益表示：

```text
超额 = 策略收益 - 买入持有收益
```

举例：

```text
策略收益：7.51%
买入持有：-12.36%
超额：19.87%
```

意思是：

- 如果用策略交易，赚了 7.51%。
- 如果简单买入并持有，亏了 12.36%。
- 策略相对买入持有多出了 19.87 个百分点。

再看另一种情况：

```text
策略收益：-11.65%
买入持有：-12.36%
超额：0.71%
```

这表示策略本身还是亏钱，但比一直拿着少亏 0.71 个百分点。

所以要记住：

```text
策略收益 > 0：策略本身赚钱
超额 > 0：策略跑赢买入持有
```

二者不是一回事。

## 7. 为什么高价股可能没有交易

当前回测按 A 股真实交易习惯，默认 100 股一手。

如果一只股票价格很高，初始资金可能买不起一手。

例如：

```text
股价：1,400 元
一手：100 股
最低买入金额：140,000 元
```

如果单票初始资金只有 100,000 元，即使出现买入信号，也无法成交。

这类情况在贵州茅台这类高价股上尤其常见。

解决方式：

```bash
uv run x-growth-backend tech-backtest --initial-cash 200000
```

提高初始资金后，策略才可能买入一手。

## 8. 三个策略怎么对比

不要只看收益率。更好的阅读顺序是：

1. 先看有没有足够交易次数。
2. 再看策略收益是否为正。
3. 再看是否跑赢买入持有。
4. 再看最大回撤是否能接受。
5. 最后看交易明细，理解策略为什么赚钱或亏钱。

常见情况解释：

| 情况 | 含义 |
| --- | --- |
| 策略收益正，超额正 | 赚钱，而且跑赢持有 |
| 策略收益正，超额负 | 赚钱了，但不如直接持有 |
| 策略收益负，超额正 | 亏钱了，但比一直持有少亏 |
| 策略收益负，超额负 | 亏钱，而且不如一直持有 |

不同指标的性格：

| 指标 | 策略风格 | 常见问题 |
| --- | --- | --- |
| MACD | 趋势跟随 | 震荡市容易反复交易 |
| KDJ | 短线波段 | 单边下跌容易过早买入 |
| RSI | 情绪修复 | 信号少，样本可能不足 |

## 9. 新手常见误区

### 误区一：胜率越高越好

不一定。胜率高但单次亏损很大，最后仍然可能亏钱。

要同时看：

```text
胜率
平均盈利
平均亏损
最大回撤
总收益
```

### 误区二：超额为正就是好策略

不一定。超额为正只说明跑赢买入持有。

如果策略收益是负的，说明它仍然亏钱，只是亏得少一些。

### 误区三：一段时间有效就长期有效

不一定。技术指标策略可能只适合某类市场环境。

例如：

- MACD 更喜欢趋势行情。
- KDJ 更喜欢震荡波段。
- RSI 更喜欢极端情绪后的修复。

所以一个策略在某一年有效，不代表下一年仍然有效。

### 误区四：忽略交易成本

交易越频繁，成本越重要。

如果一个策略频繁买卖，但单笔利润很薄，佣金和印花税可能会吃掉很多收益。

### 误区五：忽略无法成交

资金不足、停牌、涨跌停、流动性不足，都可能导致真实交易无法按回测价格成交。

当前第一版只处理了资金和 100 股整数手，没有模拟涨跌停无法成交和滑点。

## 10. 下一步可以怎么进阶

第一版策略适合学习指标本身。等理解清楚后，可以逐步加入更现实的规则：

- 加趋势过滤：例如只在收盘价高于 20 日均线时买入。
- 加止损：例如买入后跌破 8% 强制卖出。
- 加止盈：例如盈利超过 15% 后移动止盈。
- 加市场环境过滤：大盘弱势时降低交易频率。
- 加组合回测：多只股票共用一个资金池。
- 加滑点和涨跌停限制：让回测更接近真实交易。
- 加参数对比：例如 RSI 6、14、24 的表现差异。

## 11. 最后一句话

MACD、KDJ、RSI 的价值不在于“给你一个神奇买点”，而在于把市场观察变成可检验的规则。

对新人来说，最重要的是养成这个流程：

```text
先理解指标
再写清规则
然后做回测
最后看风险
```

这样你学到的就不是某个指标口诀，而是一套可以持续迭代的交易研究方法。
